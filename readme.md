# Alke Wallet 💰

Es una aplicación de billetera digital interactiva diseñada con un diseño moderno, responsivo y fácil de usar para administrar activos financieros simulados.

## 🛠️ Tecnologías Utilizadas
* **HTML5** - Estructura semántica de las vistas.
* **CSS3** - Estilos personalizados para entornos Fintech.
* **Bootstrap 5** - Diseño totalmente responsivo y adaptabilidad móvil.
* **jQuery** - Manipulación del DOM, efectos de transición y animaciones.
* **JavaScript (Vanilla)** - Lógica de negocio simulada (gestión de saldos y autenticación).

## 🖥️ Pantallas del Proyecto
* **Inicio de Sesión (`login.html`)** - Formulario de acceso controlado con credenciales fijas.
* **Dashboard Principal (`menu.html`)** - Resumen visual de saldos y acciones rápidas.
* **Depósito (`deposit.html`)** - Formulario dinámico para la recarga de fondos.
* **Enviar Dinero (`sendmoney.html`)** - Transferencias simuladas y agenda de contactos.
* **Movimientos (`transactions.html`)** - Historial detallado de ingresos y egresos.

## 🚀 Repositorio y Git Flow
El proyecto fue construido de forma progresiva utilizando control de versiones y la estructura de ramas requerida para garantizar buenas prácticas de desarrollo:
* `main` - Código estable y unificado.
* `feature/login` - Módulo de autenticación.
* `feature/depositos` - Módulo de abonos y balance.
* `feature/transacciones` - Módulo de envíos e historial.
